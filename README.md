## 실행 가이드

### Node.js

본 어플리케이션은 Readline - Promises API를 사용하므로 v17.0.0 이상의 Node.js가 필요합니다.

### 실행 가이드

> 패키지 설치

```bash
npm install
```

> 어플리케이션 실행

```bash
npm run start-step1
```
